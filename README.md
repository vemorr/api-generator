# Api Generator
*This library made for auto-generating api.ts or api.js from backend projects ready to connect for frontend*

### Install library using
```
npm install -g @vemorr/api-generator
```

## Supported backend frameworks:
- [Elysia](https://elysiajs.com/)
- [Express](https://expressjs.com/)
- [NestJS](https://nestjs.com/)
- [Fastify](https://fastify.dev/)
- [Adonis](https://adonisjs.com/)
- [Koa.js](https://koajs.com/)
- [Hono](https://hono.dev/)

```
Usage: api-generator [options]

Options:
  --ts        Generate api.ts with types
  --js        Generate api.js for project made on JS
  --elysia    Generate API for ElysiaJS
  --express   Generate API for ExpressJS
  --nestjs    Generate API for NestJS
  --fastify   Generate API for Fastify
  --adonis    Generate API for AdonisJS
  --koa       Generate API for Koa.js
  --hono      Generate API for Hono
  -h, --help  display help for command
```
